Past Projects

Flat Earth Space Program (FESP)

Flat Earth Space Program is a 2D space simulator created by Zachary Atkins, Gage Phillips, Thomas Gartman, and myself. Players have a large sandbox, planets, and Newtonian physics at their disposal as they attempt to pilot a small rocket. Features include trajectory prediction, a custom rocket-builder system, simulated liquid/solid rockets, and much more! FESP was built with python, and the pymunk (physics) and pygame libraries were heavily utilized.

See the repo on GitHub!

100Words

100Words is a mad-lib style web application that I developed alongside Gage Phillips, where users create a prompt for a story (100 words long exactly) that will be completed by users all over the globe, each user submitting a single word. This application is no longer hosted because we are poor undergraduate students, but the repo is still available!

See the repo on GitHub!
